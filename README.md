# Tiles for Origin Games

App for Windows 10 written in Visual Basic for Universal Windows Platform (UWP).

### What does the app do?

Pin Tiles of your games from Origin to the Windows 10 Start Menu, more information [in my web](https://pepeizqapps.com/app/origin-tiles/).
